# Basic Web Calculator

A clean and functional calculator built using **Vanilla JavaScript, HTML5, and CSS3**.  
This project demonstrates core web development concepts like DOM manipulation, event handling, and basic functional programming — without using any external libraries.

---

##  Features

-  **Basic Arithmetic** — Addition, subtraction, multiplication, and division  
-  **Clear & Delete** — Clear the entire display or remove the last character  
-  **Responsive Layout** — CSS Grid-based design that adapts to different screen sizes  
- **Error Handling** — Handles invalid mathematical expressions gracefully  

---

##  Tech Stack

- **HTML5** — Structure and button layout  
- **CSS3** — Styling and responsive grid system  
- **JavaScript (Vanilla JS)** — Calculator logic using `eval()` and string manipulation  

---

##  Project Structure

```
 basic-web-calculator
 ┣  index.html   # UI structure
 ┣  style.css    # Styling and layout
 ┗  script.js    # Logic and calculations
```

##  Security Note

This project uses JavaScript’s `eval()` function for simplicity in a learning environment.  
For production-level applications, using a custom math parser is strongly recommended for better security.
